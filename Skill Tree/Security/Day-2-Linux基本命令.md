# Linux系统文件结构

* bin: 可执行文件目录
* dev: 任何设备或者周边设备都以文件形式存储在这里
* media: 可移除设备
* opt: 第三方软件
* tmp: 临时文件
* home: 使用者主文件夹
* root: root 主文件夹
* proc: 虚拟文件系统
* etc: 配置文件
* usr: Unix Software Resouce 软件资源
* var: 放置经常性变动的文件


# 社会工程学 ： 凭借一个域名，你能得到什么？

1. 域名注册信息查询： 注册人，邮箱，手机等。
2. DNS查询：IP地址
3. 服务器扫描分析：服务器系统，软件环境，端口等。
4. 判断后台地址
5. 获取后台登录权限
